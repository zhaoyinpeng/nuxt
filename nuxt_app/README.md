# nuxt_app
## CSR(客户端渲染)
* 优点：
1. 服务端压力较小
2. 上手简单
* 缺点：
1. 不方便seo
2. 首屏加载慢
## SSR(服务器端渲染)
* 优点：
1. 方便seo
2. 解决首屏加载过慢问题
3. 可以生成缓存片段，生成静态化文件
* 缺点：
1. 服务端压力较大
2. 开发条件受限
3. 学习成本相对较高
